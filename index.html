About Esquire

Esquire is a multifunctional bot created in 2019 on the 10th of September. It was created by Joshua Slui with the goal of supplying servers with a new innovative system that allowed people to interact better, in ways such as hugging, kissing, cuddling and much more. The bot now provides this in both normal commands and slash commands, while also offering modules such as moderation, currency as well as utility. Currently being developed by three people known as Joshua Slui, Moksej and SoulRika.

Privacy Policy

Esquire gathers data for several commands, including:
- blacklist server/user
- currencylogs
- currency commands

Currency commands

For each currency command (except currencylogs) the guild ID, user ID and the amount of money you have is saved within our database. The guild ID ensures that the money you have is only in that specific guild, and the user ID ensures the money is tied to your user alone. Data for this command gets deleted when you use the resetbalance command.

Currency logs

the currencylogs command saves only the guild ID, and the channel ID. The guild ID only serves the purpose of keeping those currency logs to that guild, or rather server alone. The channel ID indicates which exact channel is used to log any balance changes, which is referred to as our currency logs.

blacklist

Furthermore, the data gathered for the blacklist command depends on whether it was a user, or a server that got blacklisted. These are two different categories, both serving pretty much the same purpose: Denying access to using Esquire. In case of a user being blacklisted, their ID and the reason will be saved within the database. If a server gets blacklisted, it would be the guild ID and the reason getting saved.

Data removal

You can request data deletion using the contact options listed below. However, please note that you can already reset/delete your own data in regards to currency on your/other people's servers. You can use resetbalance to delete data about yourself from our database in regards to currency, and currencylogs (without any arguments) to delete any data about currency logs in your guild.

You can not get any data in regards to blacklists deleted, unless you successfully appeal through one of our available contact options. Please do note that we do not hold any obligation towards you, and we do not have to provide you with any reason, or answer on removing your blacklist data. We have the full right to keep that data permanently and will do so until we decide otherwise.
