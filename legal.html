Privacy Policy

Upon using Esquire, you agree upon the privacy policy displayed here. Please read through it carefully.

What data we store and how we use it

All this data stored is necessary as the listed commands can not function without them.

Guild IDs - For currency commands, currencylogs and server blacklists.
User IDs - For currency commands and user blacklists.
Channel IDs - For currencylogs

Who can access the data?

Only the head developer can access this data, as well as modify it. Data does not get shared with third parties.

Can you delete my data?

It is possible to get your data deleted, or to delete it yourself:
Currencylogs - Just use the currencylogs command (no arguments) to disable currency logs and to delete the data.
Balance - Using resetbalance will delete any data in regards to currency commands about you. Note that this only deletes data for the guild you are in, data about your balance in other guilds does not get deleted, and the process should be repeated in the other guilds for that as well.

Besides that, it's also possible to contact us so we can delete all your data in the database. To do this, simply send us an email at joshuaslui0203@gmail.com, or join our support server with your user ID and/or guild ID and we will delete the data. This can however take several days, so we ask you to be patient. Please note that we will not delete data such as server blacklists and/or user blacklists unless a successful appeal was made.
